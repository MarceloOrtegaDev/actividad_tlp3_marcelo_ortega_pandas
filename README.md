# Actividad de manipulación de datos Pandas
## Cómo instalar este proyecto
- Clona este repositorio
```bash
  git clone https://github.com/MarceloOrtegaDev/actividad_tlp3_marcelo_ortega_pandas
```
- Crea el entorno virtual
```bash
  python -m venv venv
```
-Inicializa el entorno virtual e instala los requirements
```bash
  pip install -r requirements.txt
```
### Ya puedes ejecutar los casilleros de código del jupyter
